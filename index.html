Yield Data Processing

This project forms a foundational part of my thesis, focusing on the cleaning and preprocessing of raw yield monitor data collected from four farms across Canada. Unlike standard data cleaning tasks, yield data requires specialized treatment due to its spatial nature and unique noise characteristics. The workflow involved identifying and removing outliers, correcting for harvester lag, filtering based on speed thresholds, and handling overlap zones. Our approach was heavily inspired by two key studies:

  • Lyle et al.(2013), which outlines robust post-processing techniques for eliminating erroneous yield measurements in grain crops.
  • Blackmore (1999), which presents remedial correction methods for improving the reliability of yield maps.

These references provided a methodological basis for building a tailored pipeline suitable for spatially irregular and highly variable yield data.

Tools Used
  • Python, GeoPandas, QGIS, Shapely

CNN on Yield Prediction from UAV Imagery

This projects develops a Convolutional Neural Network (CNN) to predict crop yield from 7 band multispectral UAV orthomosaic tiles. On this experiements the model was evaluated on the ability to generalize under different training strategies :

  • Baseline : Train / Test on the same field.
  • Mixed : Combined training with samples from multiple fields.
  • LOFO : Train on all except one field, test field was held-out.
  • Cluster & Reverse Cluster : Train / Test split based on spatial clusters within fields to test intra-field generalization.
Tools Used
  • Python, PyTorch, Scikit-learn

Faithful Reasoning via DeepSeek

This project investigates how to improve the faithfulness and logical consistency of reasoning in small language models through Chain-of-Thought (CoT) fine-tuning and synthetic data distillation. We propose a novel framework that transfers high-quality reasoning capabilities from the larger DeepSeek-R1 model to a compact LLaMA-3B-Instruct model using LoRA-based fine-tuning and reinforcement learning (GRPO/Unsloth). By combining distillation and reward-based optimization, our method enhances the ability of smaller models to reason step-by-step while preserving factual accuracy.

Tools Used
  • DeepSeek-R1, LoRA-tuned LLaMA-3B, Unsloth
  • Reinforcement learning, custom faithfulness metrics, NLP, LLMs

Master's Thesis

Spatial Generalization of Crop Yield Prediction using UAV Imagery and Machine Learning

Spatial Generalization of Crop Yield Prediction using UAV Imagery and Machine Learning

My thesis focuses on assesement of machine learning models ability to generalize in spatially heterogeneous agricultural environments. Many studies have demonstrated promising results for crop yield prediction, but didn't give much attention to spatial variability factor. This research evaluates the performance of Linear Regression (LR) and Random Forest (RF) models using UAV-based multispectral imagery, focusing on both intra-field and inter-field generalization scenarios.

Key Contributions
  • Developed UAV-based crop yield prediction models using multispectral imagery and vegetation indices.
  • Designed and implemented spatial cross-validation strategies to rigorously assess model robustness.
  • Performed advanced spatial data cleaning, preprocessing, and analysis pipelines.
  • Conducted multi-field and intra-field evaluations across diverse agricultural environments.
  • Benchmarked model performance using RMSE, R², and Pearson correlation.
  • Applied methodology to farms across Manitoba and Alberta, Canada.
